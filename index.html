Code Category Full description
C80.0Disseminated malignancyDisseminated malignant neoplasm, unspecified — used when a patient has advanced metastatic disease with no identifiable primary or secondary site specified. Typically end-stage, terminal documentation.
C80.1Unknown primaryMalignant neoplasm, unspecified — used when the primary site is completely unknown AND at least a secondary/metastatic site is known. Do not use when both primary and secondary are unspecified.
C77–C79Secondary / metastaticSecondary malignant neoplasm codes by specific organ (e.g., C78.7 liver, C79.51 bone, C78.01 right lung). Represents cancer spread from a primary site. Sequenced first when primary is unknown.
G89.3Neoplasm-related painNeoplasm-related pain (acute or chronic). Principal code only when the encounter is specifically for pain management. The neoplasm code follows as additional. Not used when neoplasm management is the reason.
O9A.1–Pregnancy + malignancyMalignant neoplasm complicating pregnancy, childbirth, and the puerperium. Sequencing follows the neoplasm coding guideline exception described in Chapter 2 guidance.
Z85.–Personal historyPersonal history of malignant neoplasm. Used only when cancer has been completely excised with no evidence of recurrence or extension. Cannot be used when cancer is currently active.
M84.5–Pathologic fracturePathologic fracture in neoplastic disease. Sequenced first when the fracture itself is the focus of treatment. The underlying neoplasm is an additional code. When the neoplasm is the focus, neoplasm code leads instead.
Z51.0Radiation therapyEncounter for antineoplastic radiation therapy. The neoplasm code follows as additional. Do not assign as principal unless radiation is the sole documented reason for the encounter.
Z51.11ChemotherapyEncounter for antineoplastic chemotherapy. Neoplasm is coded additionally. Distinct from Z51.12 (immunotherapy) — commonly tested together to catch students who conflate the two.
Z51.12ImmunotherapyEncounter for antineoplastic immunotherapy. Separate code from chemotherapy. Biologic agents, checkpoint inhibitors, and CAR-T therapies fall here — not Z51.11.
